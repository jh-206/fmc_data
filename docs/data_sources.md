# Data Sources

This document lists important external data and software sources used by the project.
It is intended as a stable reference for authoritative URLs.

## RAWS FMC Stash

- 10h sensor data from an archive shared by Angel Farguell at WIRC.

## Synoptic

- Synoptic Weather API documentation:
  [https://docs.synopticdata.com/services/weather-api](https://docs.synopticdata.com/services/weather-api)
- Synoptic Variables documentation:
  [https://docs.synopticdata.com/services/variables](https://docs.synopticdata.com/services/variables)
  Canonical documentation for the Station Variables list exposed by the Synoptic API.
- Synoptic Networks documentation:
  [https://docs.synopticdata.com/services/networks](https://docs.synopticdata.com/services/networks)
  Canonical documentation for the Networks list exposed by the Synoptic API.
- Synoptic dataset overview:
  [https://docs.synopticdata.com/services/the-mesonet-dataset](https://docs.synopticdata.com/services/the-mesonet-dataset)
  Overview page that points to the broader station variables and network/provider reference material.
